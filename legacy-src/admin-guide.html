Course Upload Workflow

  1. Create course shell: Go to Courses, fill course title, price, summary, welcome note, and cheat sheet, then save course.
  2. Add topic: Select the course, add topic title, status, and YouTube lesson URL if available.
  3. Add lesson file: Upload an HTML file or PDF file. HTML saves inside the course data. PDF uploads to Supabase Storage.
  4. Add fallback HTML: If you do not upload a file, paste lesson HTML in the content fallback field.
  5. Add topic quiz: Upload quiz HTML or paste quiz HTML in the topic quiz field.
  6. Save topic: Watch the admin status. It should show that Supabase saved the course data.
  7. Verify: Open Courses in another browser/device and refresh with the latest site link.

Rules For Reliable Uploads

  • Use PDF for downloadable notes, worksheets, cheatsheets, and long reading material.
  • Use HTML for interactive-looking lessons, structured pages, code examples, and topic explanations.
  • Use YouTube links for video lessons; do not upload video files to Supabase for now.
  • After every save, check the Supabase saved message near the admin command center.
  • Use Publish Browser Backup only for recovery of old local content.
  • Use Publish to Cloud when you want to push the full visible admin content snapshot.

AI Prompt Templates

Use these prompts when creating course content, quizzes, and coding questions.

HTML Lesson Prompt

Use this when you want HTML for a course topic.

Create an ASG Tech course lesson as a clean HTML fragment.

Topic: [TOPIC NAME]
Level: [Beginner / Intermediate / Advanced]
Course: [COURSE NAME]
Goal: [WHAT STUDENT SHOULD LEARN]

Rules:
- Return only HTML fragment, not full html/head/body.
- Do not include script tags.
- Use semantic headings, paragraphs, lists, tables, and code blocks.
- Include: learning objectives, explanation, worked example, common mistakes, practice task, short recap.
- Keep it mobile friendly.
- Escape code examples inside pre/code blocks.
- Make the tone practical for students.
- Add a final "Next step" section.

Topic Quiz HTML Prompt

Use this for a quiz block inside a topic.

Create a short HTML quiz fragment for this lesson.

Topic: [TOPIC NAME]
Level: [LEVEL]
Question count: 5

Rules:
- Return only HTML fragment.
- Do not include scripts.
- Use h2, ol, li, details, summary.
- Each question should have 4 options.
- Put answer and explanation inside a details block.
- Include practical examples, not only definitions.

Weekly Admin Content Plan

  1. Choose one main course topic for the week.
  2. Create one HTML lesson for the topic.
  3. Add one PDF notes/resource file.
  4. Add one YouTube lesson link.
  5. Add 5 quiz questions.
  6. Add 1 coding challenge.
  7. Publish an announcement telling students what is new.
  8. Check from another device after saving.

Before You Finish

  • Course title is clear.
  • Topic title is short and searchable.
  • PDF opens after upload.
  • HTML lesson does not contain script tags.
  • YouTube link opens in the student page.
  • Quiz/coding challenge has correct answers and test cases.
  • Supabase save status confirms the update.
  • New browser/device can see the content.
