Career roadmap

Complete Roadmaps for Data & AI Careers

Choose your career path and follow a step-by-step roadmap from beginner to job-ready level.

The fields below are connected, but each role has a different focus.

Role Main Focus
Data AnalystBusiness insights & dashboards
Data ScientistPrediction, statistics & ML

ML EngineerProduction ML systems

1. Data Analyst

Goal: Turn raw data into business insights.

Basic Programming

  • Python: variables, loops, functions, OOP basics, file handling
  • Libraries: NumPy and Pandas
  • SQL: SELECT, WHERE, GROUP BY, ORDER BY, JOIN, HAVING, subqueries, window functions
SELECT department, AVG(salary)
FROM employee
GROUP BY department;

Analysis, Statistics & Visualization

  • Pandas: data cleaning, missing values, groupby, merge, pivot table
  • NumPy: arrays, matrix operations, broadcasting
  • Statistics: mean, median, mode, variance, standard deviation, probability, normal distribution, hypothesis testing, correlation
  • Visualization: Matplotlib, Plotly, Seaborn, Power BI, Tableau

Business & Portfolio

  • Excel: pivot table, VLOOKUP/XLOOKUP, conditional formatting, macros basics
  • Business: KPIs, reporting, metrics, stakeholder communication
  • Projects: Netflix analysis, e-commerce dashboard, COVID dashboard, banking customer analysis
  • Interview: SQL queries, Pandas coding, case studies, dashboard explanation
ToolsPython, SQL, Pandas, NumPy, Power BI, Tableau, MySQL, PostgreSQL, GitHub
Salary IndiaFresher Rs. 4-8 LPA | Mid Rs. 10-18 LPA | Senior Rs. 20+ LPA
2. Data Scientist

Goal: Build predictive models and extract intelligence from data.

Mathematics

  • Linear algebra: vectors, matrices, eigenvalues
  • Calculus: derivatives and gradient descent
  • Probability & statistics: Bayes theorem, distributions, hypothesis testing
theta := theta - alpha * dJ(theta) / dtheta
P(A|B) = P(B|A)P(A) / P(B)

Machine Learning & Deep Learning

  • Python, SQL, Pandas, NumPy, Matplotlib, SQL joins, window functions
  • ML: linear regression, logistic regression, decision tree, random forest, XGBoost, SVM, KNN, clustering, PCA
  • DL: ANN, CNN, RNN, LSTM, transformers with TensorFlow and PyTorch

Specialization & Projects

  • NLP: tokenization, embeddings, BERT, LLM basics
  • Computer vision: OpenCV, object detection, segmentation
  • MLOps basics: Docker, FastAPI, MLflow, deployment
  • Projects: house price prediction, churn prediction, sentiment analysis, image classification, soil moisture prediction using PINN
ToolsScikit-learn, XGBoost, PyTorch, TensorFlow, Plotly, FastAPI, MLflow
SalaryFresher Rs. 6-12 LPA | Mid Rs. 15-30 LPA | Senior Rs. 40+ LPA
3. Data Engineer

Goal: Build scalable data systems and pipelines.

Core Skills

  • SQL mastery: complex joins, CTE, stored procedures, query optimization
  • Python: ETL scripting, APIs, automation
  • Databases: PostgreSQL, MySQL, MongoDB, Cassandra

Warehousing, Big Data & Cloud

  • Warehousing: Snowflake, Redshift, BigQuery, OLTP vs OLAP, star schema, snowflake schema
  • Big data: Hadoop, Spark, PySpark, Kafka
  • Cloud: AWS, Azure, or GCP with S3, EC2, Lambda, Glue

Production Projects

  • Orchestration: Apache Airflow and Prefect
  • DevOps: Docker, Kubernetes, Jenkins, GitHub Actions
  • Projects: ETL pipeline, real-time streaming system, data lake architecture, Kafka pipeline
SalaryFresher Rs. 7-12 LPA | Mid Rs. 18-35 LPA | Senior Rs. 45+ LPA
4. ML Engineer

Goal: Deploy and scale ML models in production.

Core Difference

  • Data Scientist builds models and focuses on accuracy
  • ML Engineer deploys, scales, monitors, and improves production ML systems

Learn

  • Software engineering: DSA, APIs, OOP, system design
  • Machine learning: model training, hyperparameter tuning, evaluation
  • Deployment: FastAPI, Flask, Streamlit, Docker, Kubernetes
  • MLOps: MLflow, Kubeflow, CI/CD, monitoring

Cloud & Projects

  • Cloud: AWS SageMaker, Vertex AI, Azure ML
  • Projects: ML API deployment, recommendation system, fraud detection, real-time ML inference
SalaryFresher Rs. 8-15 LPA | Mid Rs. 20-40 LPA | Senior Rs. 50+ LPA
5. AI/ML Engineer

Goal: Build advanced AI systems using ML + DL.

Advanced Deep Learning

  • Transformers, attention mechanism, diffusion models, GANs, GNNs, PINNs
  • Frameworks: PyTorch, TensorFlow, Hugging Face

AI Domains

  • NLP: BERT, GPT, T5
  • Vision: YOLO, ViT, SAM
  • Audio: Whisper, Deepgram, speech models
  • Optimization: quantization, LoRA, fine-tuning

Projects

  • Chatbot, AI tutor, AI interviewer, medical AI system, multimodal assistant
SalaryFresher Rs. 10-18 LPA | Mid Rs. 25-50 LPA | Senior Rs. 60+ LPA
6. GenAI Engineer

Goal: Build systems using LLMs and Generative AI.

Foundation & LLMs

  • Python, APIs, ML basics, deep learning basics, transformers
  • LLMs: tokens, embeddings, attention, context window
  • Models and providers: GPT, LLaMA, Gemini, Claude, Mistral, OpenAI, Google, Anthropic, Meta
Attention(Q,K,V) = softmax(QK^T / sqrt(d_k))V

Build GenAI Systems

  • Prompting: zero-shot, few-shot, chain-of-thought, structured prompting
  • Embeddings and vector DBs: Pinecone, Weaviate, ChromaDB, FAISS
  • RAG: chunking, retrieval, re-ranking, hybrid search
  • Agents: LangChain, LangGraph, CrewAI, AutoGen

Advanced GenAI

  • Fine-tuning: LoRA, QLoRA, PEFT, RLHF basics
  • Multimodal AI: image generation, video AI, voice AI, Stable Diffusion, Whisper, Flux
  • Deployment: vLLM, Ollama, FastAPI, Docker, Kubernetes
  • Products: AI resume screener, AI coding assistant, AI PDF chatbot, AI research assistant, AI interviewer, AI teaching platform
Best StackOpenAI, Gemini, LLaMA, Mistral, LangChain, Pinecone, Docker, FastAPI, React/Next.js

Best Certifications

  • Data Analyst: Google Data Analytics, Microsoft Power BI
  • Data Scientist: IBM Data Science, Andrew Ng ML
  • Cloud: AWS Certified, Azure AI Engineer
  • GenAI: DeepLearning.AI GenAI, LangChain courses

Best Platforms to Learn

  • YouTube for free learning
  • Coursera for structured learning
  • Kaggle for practice
  • LeetCode for DSA
  • HackerRank for SQL/Python
  • Hugging Face for GenAI

Recommended Order

  • Phase 1: Python -> SQL -> Pandas -> Statistics
  • Phase 2: Visualization -> ML -> Projects
  • Phase 3: Deep Learning -> Deployment -> Cloud
  • Phase 4: LLMs -> RAG -> AI Agents -> GenAI
